Transition

animation simple : passe d'un état A à un état B

Éléments nécessaires pour créer une transition

  • une propriété CSS à modifier
  • une valeur initiale pour la propriété
  • une valeur finale pour la propriété
  • une durée
  • une pseudoclasse pour déclencher la transition (hover/active/visited)

Paramètres

version longue

img{ transition-property: all; /* all none margin color identifiant, height sliding*/ transition-duration: 2s; /* 2000ms , 7s, 8s => par ordre de propriété*/ transition-timing-function: ease; transition-delay: 0.5s; } #homer_div_1 img:hover{ transform: scale(1.5) rotate(360deg); }

version compacte

transition: <property> <duration> <timing-function> <delay>

transition: width 2s ease, height 2s linear;

Modulation de la vitesse de déroulement

linear
ease (default)
cubic-bezier()
ease-in-out
ease-in
ease-out
steps(7)
