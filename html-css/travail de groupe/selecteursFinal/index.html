Les sélécteurs CSS

Le sélécteur "+"

Exemple : "h3 + p"

Ce sélécteur va permettre de choisir le premier élément qui suit l'autre. Ici, les premiers "p" qui suivent des "h3"
Dans l'exemple ci dessus, cela séléctionnera tous les éléments "p" placés directement après un h3.

Le sélécteur "~"

Exemple : "h3 ~ p"

Séléctionne TOUS les éléments "p" qui sont précédés d'un h3. Ne doit pas être spécialement directement précédés.

Le sélécteur d'attribut "^="

Exemple : "a[href^="https"]"

Séléctionne tous les les éléments dont la valeur de l'attribut ciblé COMMENCE par *insérer une chaîne de caractères*

Dans l'exemple ci-dessous, séléction en bleu :

http://exemple1

https://exemple2

http://exemple3

https://exemple4

Le sélécteur d'attribut "$="

Exemple : "a[href$="toto"]"

Séléctionne tous les les éléments dont la valeur de l'attribut ciblé FINIT par *insérer une chaîne de caractères*

Dans l'exemple ci-dessous, sélection en bleu :

http://exempletoto

https://exemple2

http://exemple3toto

https://exemple4

Le sélécteur d'attribut "*="

Exemple : "a[href*="monkey"]"

Sélectionne tous les les éléments dont la valeur de l'attribut ciblé CONTIENT *insérer une chaîne de caractères*

Dans l'exemple ci-dessous, sélection en bleu :

http://exemplemonkeytoto

https://exemple2

http://exemple3toto

https://exemplemonkey4

La pseudo-classe ":target"

Exemple : ":target"

":target" s'appliquera à l'élément étant ciblé par une ancre de type "a", s'enlèvera si une nouvelle ancre est sélectionnée

La pseudo-classe ":not()"

Exemple : "p:not(.testNot)"

":not()" permet d'exclure certains éléments d'une sélection, ceux-ci doivent être mis entre paranthèses et peuvent être décrite par une autre séléction CSS
